Predicting Ethereum Network Gas Usage with DNN

PythonTensorFlowDNNBayesian SearchGradient-Stability Techniques

Developed a deep learning framework to predict minute-level Ethereum gas usage using on-chain and market activity data for DSO-456 Midterm Exam. Using TensorFlow/Keras, I implemented and evaluated multiple architectures from single-perceptron baselines to deep neural networks using Functional API with He initialization, experimenting with activation functions (ReLU, Leaky ReLU, Swish), gradient-stability techniques (clipping, batch normalization), and regularization methods (dropout, L1, early stopping). Hyperparameter optimization was performed via Random and Bayesian Search to refine learning rate, layer depth, and batch size. All preprocessing and training used chronologically split data to prevent leakage, with performance measured through R² and MSE. Results showed a substantial improvement from a PCA-linear baseline (R² ≈ 0.59) to the Bayesian-optimized DNN (R² ≈ 0.70), demonstrating the network’s ability to capture nonlinear dependencies in volatile crypto-economic environments.
