Predicting Ethereum Network Gas Usage with DNN

PythonTensorFlowDNNBayesian SearchGradient-Stability Techniques

Developed a deep learning framework to predict minute-level Ethereum gas usage using on-chain and market activity data for DSO-456 Midterm Exam.

Implemented multiple architectures — from single-perceptron baselines to deep neural networks using TensorFlow/Keras Functional API — with He initialization and activation experiments (ReLU, Leaky ReLU, Swish). Applied gradient-stability and regularization methods (batch normalization, dropout, L1, early stopping).

Tuned hyperparameters via Random and Bayesian Search to refine learning rate, layer depth, and batch size, using chronologically split data to avoid leakage. Results showed a strong improvement from baseline (R² ≈ 0.59) to optimized DNN (R² ≈ 0.70).
