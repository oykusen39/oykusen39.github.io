Tic-Tac-Toe AI (C++)

Minimax AlgorithmRecursive Tree TraversalDepth-First Search

Implemented a turn-based AI as a TAC-365 project using the Minimax algorithm with recursive game-tree traversal and backtracking to evaluate all move outcomes.

Used a custom tree data structure (GTNode) with dynamic memory allocation and vector-based child management to represent state transitions.

Applied depth-first search (DFS) for move exploration and score propagation for optimal decision-making.

Demonstrated understanding of recursion, tree traversal, and heuristic evaluation within an algorithmic decision framework.

Encapsulated logic through object-oriented design, separating concerns across header/source files.
